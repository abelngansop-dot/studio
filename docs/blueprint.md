# **App Name**: Inoubliable Events

## Core Features:

- Service Selection: Allow clients to select multiple event services such as photographer, videographer, caterer, and logistics.
- Event Details Input: Capture event details including city, date, time, and duration, with a non-blocking user flow.
- Booking Confirmation: Display a confirmation message upon successful submission of event details, informing the client that they will be contacted soon.
- Client Review System: Enable clients to rate services and leave comments, with admin moderation to filter offensive language.
- Admin Dashboard: Provide a central dashboard for administrators to manage user traffic, reservations, and booking statuses.
- Service Management: Allow administrators to add, modify, disable, and delete event services, including descriptions and categories.
- Client Communication Tool: A tool for suggesting targeted pop-up messages, banners, or messages based on user type or service requested. The LLM determines if the popup message is appropriate.

## Style Guidelines:

- Primary color: Violet (#9400D3) to inspire trust and festivity.
- Background color: Light lavender (#E6E6FA), a desaturated version of violet.
- Accent color: Blue (#0000FF) for icons and buttons.
- Body and headline font: 'PT Sans', a sans-serif combining modernity and warmth.
- Use clear, readable icons that can be easily understood on mobile devices.  Ensure all icons render correctly, even on low-resolution screens
- Mobile-first design with large, readable buttons and one-hand usability.
- Use fluid animations and immediate visual feedback to enhance user interaction and make the application feel responsive.
